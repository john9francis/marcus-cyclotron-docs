setBeamCurrent [BeamCurrent] [Unit]

Set the beam current for activity calculations

Range : BeamCurrent>0

Available Geant4 state(s) : PreInit Idle

Parameters
BeamCurrent type d
Unit type s Omittable : default value = microampere Parameter candidates : A mA muA nA ampere milliampere microampere nanoampere

setTimeOfIrradiation [IrradiationTime] [Unit]

Set the irradiation time for activity calculations

Range : IrradiationTime>0

Available Geant4 state(s) : PreInit Idle

Parameters
IrradiationTime type d
Unit type s Omittable : default value = hour Parameter candidates : s ms us ns ps min h d y second millisecond microsecond nanosecond picosecond minute hour day year

setBeamEnergy [BeamEnergy] [Unit]

Set the proton beam energy

Range : BeamEnergy>0

Available Geant4 state(s) : PreInit Idle

Parameters
BeamEnergy type d
Unit type s Omittable : default value = MeV Parameter candidates : eV keV MeV GeV TeV PeV meV J electronvolt kiloelectronvolt megaelectronvolt gigaelectronvolt teraelectronvolt petaelectronvolt millielectronVolt joule
