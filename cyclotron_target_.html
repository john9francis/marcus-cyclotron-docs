/cyclotron/target/

Target material control

Commands

setThickness
changeTarget
setFoilThickness
setTiEnrichment

setThickness [TargetThickness] [Unit]

Set the target thickness
Valid range: 0.1 - 2.0 mm

Usage:
Before init: /cyclotron/target/setThickness 0.5 mm
/run/initialize

After init: /cyclotron/target/setThickness 0.5 mm
/run/reinitializeGeometry

Example: /cyclotron/target/setThickness 0.5 mm

Range : TargetThickness>=0.1 && TargetThickness<=2.0

Available Geant4 state(s) : PreInit Idle

Parameters
TargetThickness type d
Unit type s Omittable : default value = mm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi

changeTarget [TargetIsotope]

Change the target isotope

Available Geant4 state(s) : PreInit Idle

Parameters

Unit type s Omittable : default value = mm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi

setTiEnrichment [TiEnrichment]

Set the enrichment of the titanium target.
Only valid if the target is in fact titanium.
Valid range: 0 - 100
Units are automatically percent (%).
Recommended range is between 50 and 100.
Usage:
Before init: /cyclotron/target/setTiEnrichment 75
/run/initialize

After init: /cyclotron/target/setTiEnrichment 75
/run/reinitializeGeometry

Range : TiEnrichment>=0.0 && TiEnrichment<=100.0

Available Geant4 state(s) : PreInit Idle

Parameters
TiEnrichment type d
